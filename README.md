# emulator-spec-advisor

A small Python tool that recommends Android-emulator settings (CPU cores, RAM,
resolution, instance count, VT) based on your PC specs.

```bash
python advisor.py            # auto-detect
python advisor.py --cores 8 --ram 16
```
