emulator-spec-advisor

This open-source tool reads your CPU core count and RAM, then suggests balanced Android-emulator settings so games run smoothly without starving Windows.

Rules of thumb

  • Assign about half your physical cores (2–4) to the emulator.
  • 2 GB RAM for ≤8 GB systems, 4 GB for 16 GB, 6 GB above.
  • Always enable VT (Intel VT-x / AMD-V) in BIOS for a big speed-up.
